# Sistema ServiMant

Este repositorio contiene el prototipo académico del sistema de gestión de reclamos y solicitudes de mantenimiento interno desarrollado para la materia Seminario de Prácticas Informáticas.

## Descripción

El sistema ServiMant tiene como objetivo centralizar la gestión de solicitudes de mantenimiento, permitiendo registrar reclamos, asignar técnicos responsables, registrar intervenciones, actualizar estados y consultar información del proceso.

## Tecnologías utilizadas

- Java
- MySQL
- JDBC
- UML
- Draw.io

## Estructura del proyecto

- `src/modelo`: clases principales del dominio.
- `src/servicio`: clases de lógica de negocio.
- `src/dao`: clases de acceso a datos y conexión JDBC.
- `src/vista`: interfaces o pantallas del prototipo.
- `src/principal`: clase principal de ejecución.
- `database`: script SQL de creación de base de datos y consultas.
- `diagramas`: diagramas utilizados en el informe.

## Base de datos

La base de datos utilizada se denomina `servimant_db` y contiene tablas para usuarios, roles, técnicos, solicitudes, categorías, prioridades, estados, sectores, asignaciones e intervenciones.
